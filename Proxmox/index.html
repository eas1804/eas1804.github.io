Если падает сеть: (вместо eno1 - указать свою сетевую карту)

/usr/sbin/ethtool -K eno1 tso off gso off

для автоматизации, добавить это в файл /etc/network/interfaces 

auto vmbr0
iface vmbr0 inet static
        address 94.130.222.29/26
        gateway 94.130.222.1
        bridge-ports eno1
        bridge-stp off
        bridge-fd 0
        post-up /usr/sbin/ethtool -K  eno1 tso off gso off
