Cli

/usr/sbin/qm
  • Oтобразить список виртуальных машин:

    qm list или cat /etc/pve/.vmlist

  • выключить VM:

    qm shutdown "ID" или qm stop "ID"

  • Запуск VM

    qm start "ID"

  • Переапуск VM

    qm stop "ID" && qm start "ID"

  • Удалить VM

    qm destroy "ID"

  • Инфлрмация о диске

    qemu-img info vm-101-disk-0.qcow2

  • Конфиг VM здесь:

    /etc/pve/qemu-server

  • Показать Конфиг :

    qm config 102

  • Диски VM здесь:

    /var/lib/vz/images/

  • Перезапустите службу pvedaemon, чтобы освежить управление VM

    systemctl restart pvedaemon

  • Перосмотр журнала pvedaemon

    journalctl -u pvedaemon -f

    journalctl -u pvedaemon | tatl -n 20
