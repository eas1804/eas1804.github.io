NTP Client

/system clock set time-zone-name=Europe/Kiev
/system ntp client servers
add address=pool.ntp.org
add address=ua.pool.ntp.org
add address=0.ua.pool.ntp.org
add address=1.ua.pool.ntp.org

Если не работает синхронизация времени через SNTP Client в Mikrotik, то во вкладке IP -> DHCP Client (далее ваше подключение) уберите галочку Use Peer NTP, после этого все будет нормально синхронизироваться. Галочка отвечает за получение времени от вашего провайдера, и если у него нет этой услуги - будете мучаться
