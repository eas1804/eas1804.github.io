Создаем профиль безопасности

Иия профиля безопасности: MY_wifi_profile, SSID:Esplanadnaya_2, Пароль: walker12 
/interface wireless security-profiles

set [ find default=yes ] supplicant-identity=Esplanadnaya_2
add authentication-types=wpa2-psk comment=walker12 eap-methods="" group-key-update=1h management-protection=allowed \
mode=dynamic-keys name=MY_wifi_profile supplicant-identity="" wpa2-pre-shared-key=walker12

Задаем параметры wifi интерфейсу 2.4 ГГЦ

/interface wireless
set [ find default-name=wlan1 ] band=2ghz-onlyn country=ukraine disabled=no distance=indoors frequency=auto \
installation=indoor mode=ap-bridge security-profile=MY_wifi_profile ssid=Esplanadnaya_2 wireless-protocol=802.11

Задаем параметры wifi интерфейсу 5 ГГЦ

/interface wireless

installation=indoor mode=ap-bridge security-profile=MY_wifi_profile ssid=Esplanadnaya_2_5Ghz wireless-protocol=802.11
