Шаг 1. Пердварительные натсройки

сбрасываем настройки маршрутизатора без создания дефолтной конфигурации

/system reset-configuration no-defaults=yes skip-backup=yes

Если нужно сохранить пользоватлей - дописываем

keep-users=yes

Задаем пользователя, пароль. Отключаем ненужные сервисы

/use add name=USER password=PASSWORD!!! group=full

/user disable admin

/ip service disable api,api-ssl,ftp,telnet,www-ssl

Задать имя компании

/system identity set name=Company
