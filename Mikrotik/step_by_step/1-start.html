Шаг 1. Предварительные настройки

  1. Сбрасываем настройки маршрутизатора без создания дефолтной конфигурации

    /system reset-configuration no-defaults=yes skip-backup=yes keep-users=yes

    Cохранить пользоватлей: keep-users=yes

  2. Задаем пользователя, пароль.

    /use add name=USER password=PASSWORD!!! group=full

    /user disable admin

  3. Отключаем ненужные сервисы

    /ip service disable api,api-ssl,ftp,telnet,www-ssl

  4. Задать имя компании

    /system identity set name=Company
