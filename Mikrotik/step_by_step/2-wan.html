Шаг 2. WAN

ether1 - смотрит в нитернет

Настройка Dynamic IP

/ip dhcp-client add comment=WAN_DHCP dhcp-options=hostname,clientid disabled=no interface=ether1 use-peer-ntp=no

use-peer-ntp=no - не использовать сервер времени провайдера

Настройка Static IP

Например IP 213.160.132.222/30, gate 213.160.132.221 заменить на свое)

255.255.255.0 /24
255.255.255.128 /25
255.255.255.192 /26
255.255.255.224 /27
255.255.255.240 /28
255.255.255.248 /29
255.255.255.252 /30

/ip address add address 213.160.132.222/30 interface ether1 comment=IP_adress_ISP

/ip route add gateway=213.160.132.221 comment=Gate

После этого должен заработать ping 8.8.8.8

DNS

/ip dns set servers=8.8.8.8,208.67.222.222

После этого должен заработать ping i.ua

Подмена mac-адреса

/interface ethernet set ether1 mac-address=00:11:22:33:44:55

где 00:11:22:33:44:55 — mac зарезервированный у провайдера.

Публичне DNS. IP-адреса

  • 8.8.8.8 и 8.8.4.4 - Google Public DNS
  • 9.9.9.9 - Quad9

    предназначен для блокировки вредоносных доменов, защиты от фишинга и вредоносного ПО. (с акцентом на кибербезопасность)

  • 1.1.1.1 и 1.0.0.1 - Cloudflare DNS

    Также есть версия 1.1.1.2 и 1.0.0.2 с фильтрацией вредоносных доменов и 1.1.1.3/1.0.0.3 с дополнительной блокировкой контента для взрослых.

  • 208.67.222.222 и 208.67.220.220 - OpenDNS (принадлежит Cisco)
  • 94.140.14.14 и 94.140.15.15 - AdGuard DNS

    DNS-сервис, ориентированный на блокировку рекламы, трекеров и вредоносных доменов. Подходит для пользователей, которые хотят уменьшить навязчивую рекламу в интернете

  • 185.228.168.9 и 185.228.169.9 - CleanBrowsing.

    DNS-сервис с акцентом на безопасность и фильтрацию контента, подходящий для семейного использования или корпоративных сетей. (фильтрация контента для взрослых)
