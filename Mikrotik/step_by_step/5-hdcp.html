DHCP

Внимательно с IP!!!!

Создаем диапазон адресов

/ip pool

add name=dhcp_pool_lan ranges=192.168.72.20-192.168.72.254

Создаем сервер DHCP

/ip dhcp-server

add address-pool=dhcp_pool_lan disabled=no interface=bridge_lan lease-time=1d10m name=dhcp_srv_lan

Задаем сеть

/ip dhcp-server network

add address=192.168.72.0/24 dns-server=8.8.8.8 gateway=192.168.72.1

Как связать ip 172.16.181.21 и MAC E6:26:3E:49:85:D8

/ip dhcp-server lease

add address=172.16.181.21 mac-address=E6:26:3E:49:85:D8 server=dhcp_srv_lan

После отключить устройчство с MAC E6:26:3E:49:85:D8 и включить

server можно увидеть в поле Name

/ip dhcp-server print

P.S. server=dhcp_srv_lan можно не указывать
