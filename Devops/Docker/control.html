Управление контейнерами

Допустим наш контейнер называется - nginx-proxy

  1. Вход в консоль контейнера - exec

    docker exec -it nginx-proxy bash

  2. Выполнить команду в контейнере

    docker exec -it nginx-proxy /usr/sbin/nginx -s reload

  3. Информация о контейнере

    docker inspect nginx-proxy

    Контейнеры хранятся в виде папок и файлов в директории /var/lib/docker/volumes
