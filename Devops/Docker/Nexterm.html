  1. ️Подключение к серверам по SSH, RDP, VNC или TELNET через браузер. Порт 6989
  2. ️Для соединений по SSH работает обзор файлов через SFTP.
  3. ️Для серверов с подключением по SSH можно включить простенький оперативный онлайн мониторинг с хранением метрик 24 часа.
  4. ️Поддерживается 2FA с помощью TOTP.
  5. ️Настроенные подключения можно раскладывать по организациям со своими настройками, учётными данными и пользователями, которым разрешён доступ.
  6. ️Все соединения логируются, есть возможность записывать сессии.
  7. ️Соединениям можно присваивать тэги для группировки.
  8. ️Можно подключить ИИ ассистента, который будет подсказывать команды в консоли. Поддерживается Ollama, ApenAI и совместимые с ним.
  9. ️Есть приложения для всех популярных систем, чтобы подключаться к серверу напрямую. Причём приложения легковесные. Под Windows всего 15 МБ весит, работает на базе WebView2.
  10. ️Проект полностью бесплатный, исходники открыты.
