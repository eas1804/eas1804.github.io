Просмотр

  1. Просмотр работающих контенеров

    docker ps

  2. Просмотр всех контенеров (работающих и остановленных)

    docker ps -a

    docker container ls --format 'table {{.Names}} | {{.Status}}'| nl -v 0

  3. Посмотреть остановленные контейнеры

    docker ps -a -f status=exited

  4. Посмотреть логи контейнера

    Допустим наш контейнер называется - nginx-proxy

    docker logs nginx-proxy или docker logs --tail 100 nginx-proxy

  5. Информация о контейнере

    docker inspect nginx-proxy

    Контейнеры хранятся в виде папок и файлов в директории /var/lib/docker/volumes

  6. Посмотреть запущенные процессы в контейнере

    docker top nginx-proxy

  7. Статистика потребляемых ресурсов контейнера или группы контейнеров

    docker stats nginx-proxy или docker stats container1 container2 container3

  8. Просмотр занимаемого места Docker

    docker system df

  9. Просмотр используемых портов

    docker port nginx-proxy
