Установка Docker

curl -o - https://get.docker.com | bash -

на всякий случай

iptables-save > /etc/iptables/docker.ruls

Дать правло пользователю запускать контейнеры

sudo usermod -aG docker youruser

Установка docker-compose

curl -L https://github.com/docker/compose/releases/download/v$(curl -Ls https://www.servercow.de/docker-compose/latest.php)/docker-compose-$(uname -s)-$(uname -m) > /usr/local/bin/docker-compose

chmod +x /usr/local/bin/docker-compose

c-top

программа для просмотра используемых ресурсов контейнерами

в ctop все метрики собраны в одно место для каждого контейнера.

Вариант 1. Запуск в docker

nano /usr/local/bin/ctop

docker run --rm -ti --name=ctop --volume /var/run/docker.sock:/var/run/docker.sock:ro quay.io/vektorlab/ctop:latest

Вариант 2. Устанока приложения

echo "deb http://packages.azlux.fr/debian/ buster main" | tee /etc/apt/sources.list.d/azlux.list

wget -qO - https://azlux.fr/repo.gpg.key | apt-key add -

apt update

apt install docker-ctop
