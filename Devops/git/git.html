Git

git config --global user.name "User Petrenko"

# имя или ник нужно написать латиницей и в кавычках

git config --global user.email username@gmail.com

Все глобальные настройки Git хранит в файле .gitconfig в домашней директории

Вывести глобальные настойки

git config --list

commit

git commit -am "changed two files"

Заменяет:

git add . git commit -m "changed two files"

status

git status

посмотреть изменения репозитория

git log

git log --oneline
