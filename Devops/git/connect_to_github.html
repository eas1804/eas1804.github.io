Connect to github

    Генерируем ключ:

    ssh-keygen -t rsa -C eas1804@gmail.com

  1. Разместить публичный ключ на Github

    Settings->SSHand GPG keys

  2. Проверить подключение к репозиторию

    ssh -T git@github.com

  3. связать свой локальный репозиторий с github

    git remote add origin https://github.com/eas1804/eas1804.github.io.git

  4. Узнать, какая ссылка активна (Просмотр удаленных репозиториев)

    git remote -v

  5. Также, убедитесь, что вы вошли в свой аккаунт GitHub

    git config credential.helper store

    Когда вы выполнили эту команду и получили пустую строку без ошибок, это означает, что команда выполнилась успешно.
