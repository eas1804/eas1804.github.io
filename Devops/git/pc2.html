Работа на несколькоих устройствах

Важно соблюдать цикл "забрал — изменил — отдал":

Перед началом работы всегда делай:

git pull

чтобы забрать изменения (например, те, что ты сделал с "zfs autotrim" на первом ПК).

После работы делай стандартно:

git add . -> git commit -m "..." -> git push

Установка Git на втором компьютере

Шаг 1: Установка Git

Если Git еще не установлен на втором компьютере:

Windows: Скачай Git for Windows

Debian:

sudo apt update && sudo apt install git

Шаг 2: Идентификация пользователя

Чтобы коммиты на втором компьютере подписывались твоим именем, выполни:

git config --global user.name "Your Name"

git config --global user.email "your_email@example.com"

Шаг 3: Настройка доступа (SSH — рекомендуемый способ)

3.1 Генерация ключа на новом компьютере:

ssh-keygen -t ed25519 -C "your_email@example.com"

3.2. Добавление ключа в GitHub:

Открой файл публичного ключа:

cat ~/.ssh/id_ed25519.pub

type C:\Users\Имя\.ssh\id_ed25519.pub).

Скопируй содержимое.

Зайди на GitHub: Settings -> SSH and GPG keys -> New SSH key.

Вставь ключ и сохрани.

3.3 Проверка связи:

ssh -T git@github.com

Шаг 4: Клонирование репозитория

Теперь создай папку, где будет лежать проект, и скачай его:

cd \путь\к\папке

git clone git@github.com:eas1804/eas1804.github.io.git
