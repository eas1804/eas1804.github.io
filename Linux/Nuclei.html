Сканер уязвимостей Nuclei

Install

wget https://github.com/projectdiscovery/nuclei/releases/download/v3.4.10/nuclei_3.4.10_linux_amd64.zip
unzip nuclei_3.4.10_linux_amd64.zip
mv ./nuclei /usr/local/bin/nuclei

Scan

nuclei -u Адрес

Вывести только low,medium,high,critical (-s)

nuclei -u Адрес -s low,medium,high,critical

Сохранить результаты в файл (-o)

nuclei Адрес -s low,medium,high,critical -o nuclei_report.txt

Задать список адресов для сканирования (-L)

nuclei -l ip-list.txt -s low,medium,high,critical -o nuclei_report.txt

Если будете сканировать сайты, веб сервера или любые другие сервисы, где есть ограничение на одновременное количество запросов, то используйте ключ -rl или -rate-limit

nmap -iL ip-list.txt -p- -T4 -oN nmap_report.txt
