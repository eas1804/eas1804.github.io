Дополнения proxmox

Репозиторий

nano /etc/apt/sources.list.d/pve-enterprise.list

Заменить pve-enterprise

deb http://download.proxmox.com/debian/pve bookworm pve-no-subscription

или в /etc/apt/sources.list

Закомментировать содержимое

nano /etc/apt/sources.list.d/ceph.list

apt update && apt upgrade -y

nftables.conf

  1. wget https://eas1804.github.io/Linux/Proxmox/nftables.conf && mv nftables.conf /etc/

  2. systemctl restart nftables.service && systemctl enable nftables.service && nft list ruleset

nft restart after reboot

cd /root && wget https://eas1804.github.io/Linux/Proxmox/nft_restart.sh && chmod +x nft_restart.sh

echo "@reboot /root/nft_restart.sh" >> /var/spool/cron/crontabs/root

NVMe info

wget https://eas1804.github.io/Linux/Proxmox/nvme_info.sh $$ chmod +x nvme_info.sh $$ ./nvme_info.sh

Установить агент на Debain

apt install qemu-guest-agent -y && systemctl start qemu-guest-agent && systemctl status qemu-guest-agent && systemctl enable qemu-guest-agent

Проверить результат на гипервизоре

qm agent vmid ping
