Оганичить ssh ipv4. AddressFamily inet

Проверить натсройку

cat /etc/ssh/sshd_config |grep AddressFamily

Внести параметр

echo "AddressFamily inet" | tee -a /etc/ssh/sshd_config

service ssh restart

Отобразить ipv6 адрес

ip -6 addr show

ip -6 addr show | grep "scope global"

cat /etc/network/interfaces | grep address | grep :

Firewall

ip6tables -A INPUT -m conntrack --ctstate INVALID -j DROP

ip6tables -A INPUT -m conntrack --ctstate ESTABLISHED,RELATED -j ACCEPT

ip6tables -A INPUT -i lo -j ACCEPT -m comment --comment "Enable Loopback"

ip6tables -A INPUT -p tcp -s 2a03:7380:254:200:e006:6eff:fe5d:a3f9/64 -m tcp --dport 22 -j ACCEPT -m comment --comment "Lesnoy"

ip6tables -P INPUT DROP

ip6tables -L INPUT -nv


service netfilter-persistent save

Разрешить ping

ip6tables -A INPUT -p icmpv6 --icmpv6-type echo-request -j ACCEPT
