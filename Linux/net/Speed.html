Пример правильного написания скоростей передачи данных:

- килобиты в секунду — кбит/с (kbps, kbit/s или kb/s)
- мегабиты в секунду — Мбит/с (Mbps, Mbit/s или Mb/s)
- гигабиты в секунду — Гбит/с (Gbps, Gbit/s или Gb/s)

Килобиты (kb) всегда в сокращениях пишутся с маленькой буквы, хотя килобайты с большой - KB. 

Пример правильного написания размера данных:
- Килобайт — КБ, Кбайт (KB)
- Мегабайт — МБ, Мбайт (MB)
- Гигабайт — ГБ, Гбайт (GB)

iftop -i eth0

2. Для отображения статистики по отдельным хостам используйте опцию "-F" (filter):

iftop -i eth0 -F 192.168.0.1/24

3. Для вывода статистики в более удобном формате можно воспользоваться опцией "-B" (burst mode). Эта опция позволяет выводить статистику не непрерывно, а только при появлении трафика в сети:

iftop -i eth0 -B

4. Для отображения суммарной статистики использования сетевого интерфейса воспользуйтесь опцией "-t" (totals):

iftop -i eth0 -t
