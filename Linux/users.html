Users and groups

  1. Добавление пользователя Prtro с паролем password

    useradd -p $(openssl passwd -6 "password") "Prtro" -s /bin/bash -m -G sudo

    -p $(openssl passwd -1 "password"):
    Эта часть команды устанавливает пароль для нового пользователя.
    openssl passwd -1, чтобы создать хэш пароля с использованием алгоритма MD5.
    openssl passwd -6 указывает OpenSSL использовать SHA-256 для создания хэша пароля
    Этот подход считается более безопасным по сравнению с MD5. Заметьте, что не все системы могут поддерживать SHA-256 в данном контексте

  2. Добавление пользователя

    useradd user_name

  3. Добавление пользователя, с указанием каталога и оболочки

    name=mtir

    useradd -m -d /mnt/raid_md127/$name -s /bin/bash $name

    passwd $name

  4. Изменение пароля

    passwd user_name

  5. Удаление пользователя

    userdel user_name

  6. Заблокировать пользователя

    passwd -l user_name

  7. Разблокировать пользователя:

    passwd -u user_name

  8. Посмотреть статус пользователя:

    passwd -S user_name (S- заглавная)


    - P или PS: пароль установлен (пользователь разблокирован)
    – L или LK: пользователь заблокирован
    – N или NP: пароль не требуется пользователю

  9. Добавиить возможность переключиться в root

    usermod -aG sudo username

  10. Просмотр списка пользователей

    cat /etc/passwd

  11. Создать группу

    groupadd new-group

  12. Изменить имя группы test-group на имя named-group

    groupmod -n named-group test-group

  13. Удаление группы

    groupdel test-group

  14. Добавить пользователя в новую группу

    gpasswd -a new-user test-group

  15. Указать группу new-group в качестве основной группы пользователя test-user

    usermod -g new-group test-user
