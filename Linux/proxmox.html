  • Oтобразить список виртуальных машин:

    /usr/sbin/qm list или cat /etc/pve/.vmlist

  • выключить VM:

    qm shutdown "ID" или qm stop "ID"

  • Запуск VM

    qm start "ID"

  • Удалить VM

    qm destroy "ID"

  • Инфлрмация о диске

    qemu-img info vm-101-disk-0.qcow2

  • Конфиг фиртуалок ледит здаесь

    /etc/pve/qemu-server

  • Диски находятся здесь:

    /var/lib/vz/images/
