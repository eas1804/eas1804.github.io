View

column

Эта утилита делает одну вещь: выстраивает данные в удобочитаемые таблицы, используя различные разделители. В общем случае разделителем считается пробел, но его через ключ можно переназначить.

mount | column -t

column -s ":" -t /etc/passwd

column -s ":" -t /etc/passwd | awk '{print $1}'

bat

аналог cat и less одновременно

apt install bat -y

batcat /var/log/syslog

Короткий файл она целиком выводит в терминал с подсветкой и нумерацией строк, а длинный, который не умещается на экран, подгружает по мере прокрутки вниз, как less
