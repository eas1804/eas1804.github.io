Microsoft Outlook

Перенети базу на другой диск

1. Открыть консоль от имени администратора

2. Перенесение папки данных MS Outlook в новое место

move "%USERPROFILE%\Local Settings\Application Data\Microsoft\Outlook" "d:\Mail"

3. Создание соединения для папки в Windows 10

mklink /j "%USERPROFILE%\Local Settings\Application Data\Microsoft\Outlook" "d:\Mail"

Поисковая система Windows отключена.

При использовании функции мгновенного поиска Outlook не сможет выводить результаты быстрого поиска, пока этот компонент Windows не будет запущен.

Проверить состояние службы

sc query WSearch

если слуюжа не запущена, спуcтить ее

Set-Service -Name WSearch -StartupType Automatic

Start-Service -Name WSearch

После того, как служба "Windows Search" будет запущена, убедитесь, что индексирование работает правильно.

Панель управления->Крупные значки->Параметры индексирования

Убедитесь, что нужные папки и диски включены в индексирование.
Если необходимо, добавьте новые места для индексирования, нажав кнопку "Изменить".

В "Параметрах индексирования" нажмите "Дополнительно", затем в разделе "Устранение неполадок" нажмите "Перестроить".
