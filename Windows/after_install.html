Windows. After install

  1. Задать часовой пояс Украина/Киев

    Set-TimeZone -Id "FLE Standard Time"

    узнать текущий часовой пояс Get-TimeZone Просмотреть коды часовых поясов tzutil /l

  2. Отключить cortana

    Get-appxpackage -allusers Microsoft.549981C3F5F10 | Remove-AppxPackage

  3. Электропитание — максимальная производительность

    powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61

  4. Убираем экран ‘Завершение настройки вашего устройства’ в Windows 10/11

    win+r ms-settings:notifications

    Отключите опцию "Suggest ways I can finish setting up my device to get the most out of Windows"
    (Предложить способы завершения настройки устройства для наиболее эффективного использования Windows).

    или через реестр

    reg add HKCU\Software\Microsoft\Windows\CurrentVersion\UserProfileEngagement /v ScoobeSystemSettingEnabled /t REG_DWORD /d 0 /f
