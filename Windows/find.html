Поиск cmd

dir E:\*.ZBF /S /B

E:\*.ZBF — ищет все файлы с расширением .ZBF на диске E.

/S — поиск во всех подкаталогах.

/B — вывод только полного пути без лишней информации.

С выводом размеров и дат

dir E:\*.ZBF /S

Исключить каталог из поиска

dir E:\*.ZBF /S /B | findstr /V /I "\\ZBF_local\\"

Если нужно обработать каждый файл (например, скопировать)

forfiles /P E:\ /M *.ZBF /S /C "cmd /c echo @path"

/P E:\ — стартовый путь.

/M *.ZBF — маска файлов.

/S — рекурсивный поиск.

/C "cmd /c echo @path" — действие для каждого найденного файла (здесь просто вывод пути).

Совет: если файлов много, можно перенаправить результат в текстовый файл:

dir E:\*.ZBF /S /B > C:\list_zbf.txt
