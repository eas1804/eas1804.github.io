Ремонт

  1. sfc

    sfc /scannow

    запустить тест только без ремонта.

    Восстановления поврежденных файлов образа Windows

    sfc /verifyonly

  2. dism

    Восстановить поврежденные файлоы образа Windows

    dism /Online /Cleanup-Image /RestoreHealth

    только анализирует образ и предоставляет информацию о состоянии целостности системных файлов.

    DISM /Online /Cleanup-Image /ScanHealth

    Выполняет проверку целостности образа Windows, но она может быть менее полной и интенсивной по сравнению с /ScanHealth

    DISM /Online /Cleanup-Image /CheckHealth

    /RestoreHealth пытается восстановить поврежденные файлы,

    /ScanHealth просто сканирует их, а

    /CheckHealth выполняет более легкую проверку целостности.

  3. Перезапустить службу печати (Print Spooler)

    net stop spooler

    net start spooler

    ps

    Restart-Service spooler

    status

    sc query spooler

  4. defrag

    defrag F: /U /V

    /U Показывает ход выполнения операции в реальном времени. Вы будете видеть прогресс прямо в консоли.

    /V Выводит подробный отчет после завершения дефрагментации, включая статистику фрагментации и действия, которые были выполнены.

  5. Очисткв свободного пространства

    sdelete -z F:

    download Sysinternals от Microsoft.

    -z (zero free space) — этот параметр заполняет всё свободное пространство на указанном диске нулями.

    Это помогает удалить остатки удалённых файлов, которые могут быть восстановлены с помощью специализированного ПО. После выполнения команды восстановление данных с незанятого пространства становится практически невозможным.
