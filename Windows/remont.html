Ремонт

  1. sfc

    sfc /scannow

    запустить тест только без ремонта.

    Восстановления поврежденных файлов образа Windows

    sfc /verifyonly

  2. dism

    Восстановить поврежденные файлоы образа Windows

    dism /Online /Cleanup-Image /RestoreHealth

    только анализирует образ и предоставляет информацию о состоянии целостности системных файлов.

    DISM /Online /Cleanup-Image /ScanHealth

    Выполняет проверку целостности образа Windows, но она может быть менее полной и интенсивной по сравнению с /ScanHealth

    DISM /Online /Cleanup-Image /CheckHealth

    /RestoreHealth пытается восстановить поврежденные файлы,

    /ScanHealth просто сканирует их, а

    /CheckHealth выполняет более легкую проверку целостности.
