Запуск произвольного приложения в виде RemoteApp в Windows

Сервер

Нужен Windows server или win10/11 Pro или Enterprise

На сервере выполнить. После чего серервер перезагрузить

reg add "HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows NT\Terminal Services" /v fAllowUnlistedRemotePrograms /t REG_DWORD /d 1

Клиент

На клиенте содать файл rdp.

Открыть его в болкноте и диписать: 

remoteapplicationmode:i:1
RemoteApplicationName:s:КАК ДОЛЖЕН НАЗЫВАТЬСЯ ЗАГОЛОВОК ОКНА ПРОГРАММЫ В ПАНЕЛИ
RemoteApplicationProgram:s:"ПУТЬ К ИСПОЛЬНЫЕМОМУ ФАЙЛУ ПРИЛОЖЕНИЯ НА RDP СЕРВЕРЕ"
DisableRemoteAppCheck:i:1
Prompt for Credentials on Client:i:0
Alternate Shell:s:rdpinit.exe

Например 

remoteapplicationmode:i:1
RemoteApplicationName:s: AnyDesk
RemoteApplicationProgram:s:C:\Users\exch_adm\Desktop\AnyDesk.exe
DisableRemoteAppCheck:i:1
Prompt for Credentials on Client:i:0
Alternate Shell:s:rdpinit.exe
